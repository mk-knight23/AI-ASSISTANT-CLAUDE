Core Capabilities

Everything Claude Code Can Do

A complete agentic coding environment built around your terminal workflow.

Agent Loops

Claude reads, writes, executes, observes output, and self-corrects — fully autonomous multi-step task completion.

Hooks System

PreToolUse, PostToolUse, Stop, and Notification hooks for complete lifecycle automation — auto-format, lint, test, commit.

Skills (Slash Commands)

/commit, /review-pr, /tdd, /ui-ux-pro-max, /octo — 50+ reusable workflow commands invokable from the terminal.

MCP Servers

20+ MCP integrations: GitHub, Gmail, Linear, Playwright, Filesystem, Figma, Slack, Memory, Firecrawl, and more.

CLAUDE.md

Project-specific AI instructions persisted in CLAUDE.md — coding standards, architecture rules, testing requirements.

Multi-Agent Teams

Spawn parallel sub-agents via claude-teams MCP. Security audit, performance review, and testing running simultaneously.

Extended Thinking

Reserve up to 31,999 tokens for deep reasoning. Toggle with Option+T. Essential for complex architectural decisions.

Git Integration

Full git workflow — smart commits, PR creation, branch management, and conflict resolution via natural language.

Memory System

Persistent memory across sessions. Project notes, architectural decisions, and preferences stored in ~/.claude/projects/*/memory/

In Action

Real Commands I Use Daily

~/my-project — claude
# Install Claude Code
$ npm install -g @anthropic-ai/claude-code
✓ Claude Code installed

# Start interactive mode
$ claude
Claude Code v1.x.x — Type a task or /help

# Autonomous feature implementation
claude> implement JWT authentication with refresh tokens, add unit tests, update README
✓ Planning...
✓ Created src/auth/jwt.ts
✓ Created src/auth/refresh.ts
✓ Created tests/auth.test.ts
✓ All 12 tests passing
✓ README updated

# Use a skill
claude> /commit
✓ feat: add JWT authentication with refresh tokens

# Run with extended thinking
$ claude --model opus "design the microservices architecture for this monolith"
→ Using claude-opus-4-6 with extended thinking (31,999 tokens)

# Spawn parallel agents
claude> launch 3 parallel agents: security audit, performance review, test coverage
✓ Agent 1: Security analysis — in progress
✓ Agent 2: Performance review — in progress
✓ Agent 3: Coverage analysis — in progress

Real Development Patterns

01
Feature Development

claude "create CLAUDE.md" → /tdd → implement → /review-pr → /commit → push

02
Bug Fix

claude "reproduce bug" → claude "identify root cause" → claude "fix + add regression test" → /commit

03
Code Review

claude "launch 3 agents: security, performance, quality" → /review-pr → iterate

04
Architecture

claude --model opus "design architecture" → plan → /tdd → build incrementally

05
Parallel Tasks

3 agents via claude-teams → auth refactor + test suite + docs update → all done in parallel

06
Daily Standup

claude "review yesterday's commits" → "suggest today's top 3 priorities" → start first task
